Keybindings
===========

These are the default key bindings:

- Go To Step Function

  - Linux ``Super+G``
  - OSX ``Alt+G``
  - Windows ``Ctrl+Alt+Shift+G``

- Generate Step Function

  - Linux ``Super+S``
  - OSX ``Alt+S``
  - Windows ``Ctrl+Alt+Shift+S``

- Generate Missing Step Functions

  - Linux ``Super+F``
  - OSX ``Alt+F``
  - Windows ``Ctrl+Alt+Shift+F``

- Run behave

  - Linux ``Super+T``
  - OSX ``Alt+T``
  - Windows ``Ctrl+Alt+Shift+T``
